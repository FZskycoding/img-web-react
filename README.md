# 圖片搜尋網站

這是一個使用 React 開發的圖片搜尋網站，透過整合 Pexels API 提供圖片搜尋的服務。

## 功能特點

- 圖片搜尋功能
- 瀑布流式圖片展示
- 載入更多圖片功能
- 響應式設計
- 路由系統，包含首頁和關於頁面

## 使用的技術

- **React.js** - 前端框架
- **React Router** - 路由管理
- **Axios** - API 請求
- **SCSS** - 樣式管理

## 我學到甚麼?
- React 基礎概念
- 使用useState狀態管理
- React Router 的使用
- 利用Blob物件完成圖片下載功能
